#### 이번에는 ```if``` 문에 ```and```,```or```을 넣어 코드를 작성해보겠다. 
#### 예를 들어 문자열 ```p```가 있다고 할때 
```py
p = 'apple'
```
#### 다음과 같이 중첩된 if 문을
```py
if 'a' in s:
  if 'b' in 'apple':
    print('apple에는 a도 있고 b도 있어요!')
```
#### ```and```를 써서  다음과 같이 바꿀 수 있다.
```py
if 'a' in p and 'b' in p:
  print('apple에는 a도 있고 b도 있어요!')
```
#### 위처럼 중첩되었던 ```if``` 문을 한번의 ```if```문으로 줄일수 있다.
